from IPython.display import display, Latex

x = sp.Symbol("x")
def norm(p1, p2):
    p = sp.expand(p1 * p2 * (x ** 0.5))
    return sp.integrate(p, (x, 0, 1))

vs = [x**i for i in range(4)]

5.7. The line is of the form $ax + b$. The least squares line will then minimize $$\int\limits_{0}^{1} (f(x) - (ax + b))^2 dx$$ To minimize this we can use Lagrange multipliers. Just kidding, we're going to take the partial derivatives with respect to $a$ and $b$, and the minimum will occur when those are equal to zero. The partial derivative with respect to $a$ gives the equation \begin{align*} 0 &= \frac{\partial}{\partial a} \int\limits_{0}^{1} (f(x) - (ax + b))^2 dx\\ &= \int\limits_{0}^{1} 2[f(x) - (ax + b)](-x) dx\\ &= \int\limits_{0}^{1} 2[-x^{1.5} + ax^2 + bx]dx\\ &= -\frac45 + \frac{2a}{3} + b \end{align*} The partial derivative with respect to $b$ gives \begin{align*} 0 &= \frac{\partial}{\partial a} \int\limits_{0}^{1} (f(x) - (ax + b))^2 dx\\ &= \int\limits_{0}^{1} 2[f(x) - (ax + b)] dx\\ &=\int\limits_{0}^{1} 2[x^{0.5} - ax - b] dx\\ &= \frac43 - a - 2b \end{align*}

Solving for $a$ and $b$ we have $$a=\frac{4}{5}, b=\frac{4}{15}$$ so the least squares line is $$f(x) \approx \boxed{\frac{4}{5}x + \frac{4}{15}}.$$ Here's a graph of it:

5.15. First to prove equation 5.16 in the textbook. We want to set the partial derivatives with respect to each $a_r$ equal to zero. Consider any such $a_k$. We have \begin{align*} \frac{\partial}{\partial a_k}\int\limits_{a}^{b}[f(x) - \sum_{r=0}^{n}a_r\psi_r(x)]^2\text{d}x &= 0\\ &\Longleftrightarrow\\ \int\limits_{a}^{b}2[f(x) - \sum_{r=0}^{n}a_r\psi_r(x)](-\psi_k(x))\text{d}x &=0\\ &\Longleftrightarrow\\ \int\limits_{a}^{b}2f(x)\psi_k(x)\text{d}x - \sum_{r=0}^{n}2a_r\int\limits_{a}^{b}\psi_k(x)\psi_r(x)\text{d}x &= 0 \end{align*} However for $r\ne k$ we have $$\int\limits_{a}^{b}\psi_k(x)\psi_r(x)\text{d}x = 0$$ so everything cancels in the summation except the $r=k term$. We're left with $$\int\limits_{a}^{b}2f(x)\psi_k(x)\text{d}x - 2a_k\int\limits_{a}^{b}\psi_k(x)^2\text{d}x$$ $$\Longleftrightarrow$$ $$\int\limits_{a}^{b}f(x)\psi_k(x)\text{d}x = a_k\int\limits_{a}^{b}\psi_k(x)^2\text{d}x$$ Which is equation $5.16$.

So we have \begin{align*} \int\limits_{a}^{b}[f(x) - \sum_{r=0}^{n}a_r\psi_r(x)]^2\text{d}x &=\\ \int\limits_{a}^{b}[f(x)]^2\text{d}x - 2\int\limits_{a}^{b}f(x)\sum_{r=0}^{n}a_r\psi_r(x)\text{d}x + \int\limits_{a}^{b}[\sum_{r=0}^{n}a_r\psi_r(x)]^2\text{d}x &= \end{align*}

If you expand out the term inside the last integral you get $$\sum_{r=0}^n(a_r\psi_r(x))^2 + 2\sum_{j\ne k}a_ja_k\psi_j(x)\psi_k(x)$$ Because $\psi_j$ and $\psi_k$ are orthogonal for $j\ne k$, when you take the integral these terms will become equal to $0$. Now look at the second integral. From equation $5.16$ we have $$\int\limits_{a}^{b}f(x)\psi_r(x)\text{d}x = \int\limits_{a}^{b}a_r\psi_r(x)^2\text{d}x$$ So $$\int\limits_{a}^{b}f(x)\sum_{r=0}^{n}a_r\psi_r(x)\text{d}x = \int\limits_{a}^{b}\sum_{r=0}^{n}a_r^2\psi_r(x)^2\text{d}x$$

Now going back to the whole equation, we have that \begin{align*} \int\limits_{a}^{b}[f(x)]^2\text{d}x - 2\int\limits_{a}^{b}f(x)\sum_{r=0}^{n}a_r\psi_r(x)\text{d}x + \int\limits_{a}^{b}[\sum_{r=0}^{n}a_r\psi_r(x)]^2\text{d}x &=\\ \int\limits_{a}^{b}[f(x)]^2\text{d}x - 2\int\limits_{a}^{b}\sum_{r=0}^{n}a_r^2\psi_r(x)^2\text{d}x + \int\limits_{a}^{b}\sum_{r=0}^{n}a_r^2\psi_r(x)^2\text{d}x &=\\ \int\limits_{a}^{b}[f(x)]^2\text{d}x - \int\limits_{a}^{b}\sum_{r=0}^{n}a_r^2\psi_r(x)^2\text{d}x &=\\ \int\limits_{a}^{b}[f(x)]^2\text{d}x - \sum_{r=0}^{n}a_r^2\int\limits_{a}^{b}\psi_r(x)^2\text{d}x &= \end{align*} $\blacksquare$

5.17. The Fourier series coefficients are given by

$$a_0 = \frac{1}{\pi}[\int_{-\pi}^{0} -xdx + \int_{0}^{\pi} xdx] = \pi$$$$a_n = \frac{1}{\pi}[\int_{-\pi}^{0} -x\cos(nx)dx + \int_{0}^{\pi} x\cos(nx)dx] = \frac{2}{\pi}\int_{0}^{\pi} x\cos(nx)dx$$

Using integration by parts we have

$$\int x\cos(nx)dx = \frac{x\sin(nx)}{n} - \int\frac{sin(nx)}{n}dx = \frac{x\sin(nx)}{n} + \frac{\cos(nx)}{n^2}$$

so

$$a_n = \frac{2}{\pi}\cdot \frac{\cos(\pi n)-1}{n^2}$$

which is $0$ if $n$ is even and $-\frac{4}{\pi n^2}$ if $n$ is odd.

And the coefficients for $b_n$ are given with

$$b_n = \frac{1}{\pi}[\int_{-\pi}^{\pi}|x|\sin(nx)dx] = 0$$

as $|x|\sin(nx)dx$ is odd.

Putting it all together, we have

\begin{align*} |x| &= \frac{a_0}{2} + \sum a_n\cos(nx) + \sum b_n\sin(nx)\\ &= \frac{\pi}{2} - \sum_{n\text{ odd}} \frac{4}{\pi n^2}\cos(nx) \end{align*}
