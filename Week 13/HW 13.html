14.7.5¶

Letting $\mu = \frac{a^2 \Delta t^2}{h^2}$ the difference method becomes

$$u_j^{(k+1)} = (2-2\mu)u_j^{(k)} + \mu u_{j-1}^{(k)} + \mu u_{j+1}^{(k)} - u_j^{(k-1)}$$

And so

$$u^{(k+1)} = Au^{(k)} - u^{(k-1)}$$

where $A$ is the tridiagonal matrix with diagonals $\mu, 2-2\mu, \mu.$

Code is below, written in Python 3 with numpy and matplotlib.

14.7.6¶

I'll use the Fourier stability analysis. Assume that $$u_j^{(k)} = \sum g_{\xi}^k e^{ijh\xi}.$$

If $|g| < 1$ for all frequencies $\xi$ then it will be stable. Looking at just one frequency, and plugging it into the Lax-Friedrichs difference method, we see that

$$g^k e^{ijh\xi}\left[g - \frac{a\lambda - 1}2\left(e^{-ih\xi} + e^{ih\xi}\right)\right] = 0$$

Or,

$$g = (a\lambda - 1)\cdot \frac{e^{-ih\xi} + e^{ih\xi}}{2} = (a\lambda - 1)\cos(h\xi)$$

Therefore we need $|a\lambda - 1| < 1\implies \lambda\in \left(0, \frac2a\right)$

12.3.2¶

Consider the polynomial

$$\det(A -\lambda I) = 0$$

The roots of this polynomial are the eigenvalues of $A$. However, as $A$ is real, the coefficients of this polynomial are also real. It is well known that a polynomial with real coefficients can only have complex roots if they come in conjugate pairs (i.e. if $x\in \mathbb{C}$ is a root, then so is $\overline{x}$, it's complex conjugate)*.

Therefore, any complex eigenvalue of $A$ has at least one other eigenvalue with the same magnitude, and there can't be a strict inequality $|\lambda_1| > |\lambda_2|$.

* Notice that $$\overline{xy} = \overline{x}\cdot \overline{y}$$ and $$\overline{x+y} = \overline{x} + \overline{y}$$ therefore if $$p(x) = a_0 + a_1x + a_2x^2 + \cdots$$ then $$p(\overline{x}) = a_0 + a_1\overline{x} + a_2\overline{x}^2 + \cdots = \overline{a_0 + a_1x + a_2x^2 + \cdots} = \overline{p(x)}.$$

Then if $p(x) = 0,$ $$p(\overline{x}) = \overline{p(x)} = 0$$ also.

12.3.3¶

Take the shift $\mu = \lambda_1$ so that the shifted eigenvalues are

$$2\lambda_1, 0, \lambda_3+\lambda_1, \lambda_4+\lambda_1\dots.$$

By the triangle inequality, $$|\lambda_k + \lambda_1| \le |\lambda_k| + \lambda_1 < \lambda_1 + \lambda_1=2\lambda_1,$$

so the largest eigenvalue has strictly larger magnitude than all the other eigenvalues.
